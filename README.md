# 🕌 القرآن الكريم - Complete Digital Mushaf

A beautiful, responsive web application to read the complete Holy Quran (604 pages, Madinah Mushaf).

## 📦 Package Contents

This repository contains:

1. **quran-mushaf.html** - Basic beautiful Quran viewer
2. **quran-mushaf-customizable.html** - Advanced version with settings
3. **download-helper.html** - Browser tool to download & convert Quran data
4. **convert.js** - Node.js script for command-line conversion
5. **INSTRUCTIONS.md** - Detailed setup guide
6. **README.md** - This file

## ✨ Features

### Basic Version (quran-mushaf.html):
- ✅ 604 pages of complete Quran (Madinah Mushaf)
- ✅ Beautiful Arabic typography (Amiri Quran font)
- ✅ Decorative Surah headers
- ✅ Verse numbers in circular badges
- ✅ Page navigation (previous/next)
- ✅ Jump to specific page
- ✅ Complete Surah index (114 surahs)
- ✅ Keyboard shortcuts (← →)
- ✅ Responsive design for mobile
- ✅ Works offline

### Advanced Version (quran-mushaf-customizable.html):
All basic features PLUS:
- ⚙️ Customizable font family (3 Arabic fonts)
- 📏 Adjustable font size (20-40px)
- 📐 Line height control
- 🎨 Custom colors (text, background, border)
- 🌞 Light theme
- 🌙 Dark theme
- 📜 Sepia theme
- 💾 Saves your preferences
- 📱 Fully responsive

## 🚀 Quick Start (3 Steps)

### Step 1: Download Quran Data
Open `download-helper.html` in your browser and follow the instructions to download and convert the Quran data file.

### Step 2: Get the JSON file
The helper will automatically download `quran-data.json` for you.

### Step 3: Open the Mushaf
Place `quran-data.json` in the same folder as the HTML files and open either:
- `quran-mushaf.html` (basic version)
- `quran-mushaf-customizable.html` (advanced version)

## 🆘 Troubleshooting

**Problem**: Can't see Quran text  
**Solution**: Ensure `quran-data.json` is in the same folder as HTML files

**Problem**: Download helper not working  
**Solution**: Try the convert.js script instead (requires Node.js)

**Problem**: Verse numbers not showing  
**Solution**: Check that verse markers (۝) are in the data
